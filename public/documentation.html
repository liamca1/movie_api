myFlix App Documentation

Web application that provides users with access to information about different movies, directors, and genres. Users are able to sign up, update their personal information, and create a list of their favourite movies.

Return data about a single movie by title /movies/[title] GET :title none JSON object holding all data about the selected movie (description, year, genre, director, cast & img URL. e.g.

                {
                    name: 'Drama'
                    description: '',
                }
Return data about a director by name /movies/directors/[director] GET director none JSON object holding data about the selected director

                    death: '-'
                    bio: '',
                }
Allow new users to register /users POST none A JSON object holding data about the new user. e.g. 
                            
                {
                    username: "username123"
                    email: "username123@hotmail.com"
                    password: "Similarly>December4>Pantry"
                    date of birth: "01/05/1996"
                    favorites: [
                    "Apocalypse Now"
                    ]
                }
A JSON object holding data about the added user including ID. e.g. 
                            
                {
                    id: '1'
                    username: "username123"
                    email: "username123@hotmail.com"
                    password: "Similarly>December4>Pantry"
                    date of birth: "01/05/1996"
                    favorites: [
                    "Apocalypse Now"
                    ]"
                }
Allow users to update their user info (username) /users/[username] PUT :username JSON object holding the new username. e.g. 
                            
                                {
                                    "username": "NewUserName321"
                                }
JSON Object with the updated user profile. e.g. 
                            
                {
                    id: "1"
                    username: "NewUserName321"
                    email: "username123@hotmail.com"
                    password: "Similarly>December4>Pantry"
                    birthday: "01/05/1996"
                    favourites: [
                            'Apocalypse Now'
                    ]
                }
Allow users to add a movie to their list of favourites (showing only a text that a movie has been added) /users/[username]/favourites POST :username, :movie none textual message that the movie was added
Allow users to remove a movie from their list of favourites /users/[username]/[movie] DELETE :username, :movie none textual message that the movie was deleted
Allow existening users to deregister /users/[username] DELETE :username none textual message that the user account was deregistered
